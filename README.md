## Multiple React Apps
_Run the command below..._
```bash
git clone https://github.com/yoda-libs/glaze-ui-examples
cd glaze-ui-examples/multiple-react-apps
npm install
npm run bootstrap
npm start
```
_..and open `http://localhost:9000`_
